Books Tracker

Приложение позволяет сохранить свой погресс в чтении книг и делится им с друзьями, что повышает мотивацию за счет социальной составляющей. Командный проект на стеке Java/Spring Boot/JOOQ. Инфраструктура GitLab для хранения репозитория, развертывание с помощью GitLab CI/CD в k8s кластер.

Я полностью переработал систему отслежования прогресса, сделав ее более гибкой и наглядной. Разработал интеграцию с telegram: каждый день в определенное время система публикует сводку действий пользователей (информацию о прочитанных главах, комментарии). Реализовал поддержку markdown в комментариях на backend.

Программа для запоминания て формы

Обучающая программа на Java. На каждом шаге берет случайный глагол из списка и образует для него て форму (базовая форма глагола в японском языке).

Алгоритм состоит из двух частей: первая часть опеределяет спряжение глагола, вторая - образует форму по правилам. Для обеспечения корректной работы частей (модулей) программы в ходе разработки применялся подход TDD (Test Driven Development).
